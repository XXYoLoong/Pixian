# Pixian UI/UX 设计说明

## 设计关键词

东方神秘、墨迹星图、命盘、天书、灵感召唤、长篇一致性、开源工程、创作驾驶舱。

## 页面结构

1. 首页首屏：以“请笔落纸，万象成书”为主视觉，使用动态命盘与浮动信息面板建立记忆点。
2. 创作驾驶舱：左侧导航、中间故事命盘、右侧一致性巡检，避免普通后台卡片堆叠。
3. 世界观工坊：核心法则、境界阶梯、地图切片并列展示。
4. 势力星图：用星图表现数百势力，hover 展示势力名称。
5. 大纲与章节：章节列表、正文编辑器、检查墙三栏联动。
6. 开源区域：展示 CLI 命令、模块结构和发展路线。

## 动效设计

- 首屏命盘缓慢浮动与旋转。
- 符文节点呼吸发光。
- 命盘连线透明度循环变化。
- 势力星点 hover 放大并显示名称。
- 页面元素进入视口时渐入上浮。
- 操作反馈使用“灵感汇聚中”等沉浸式 Toast。

## 视觉系统

- 暗色模式：深墨黑、幽蓝、暗紫、金色点缀。
- 纸色模式：暖纸白、墨色文字、暗金强调。
- 组件风格：玻璃拟态、发光边缘、低透明噪声纹理。

## 后续可扩展

- 接入真实后端生成接口。
- 增加人物关系图谱编辑器。
- 增加地图 SVG/Canvas 绘制。
- 增加章节版本对比。
- 增加 RAG 长文记忆和一致性校验报告。
